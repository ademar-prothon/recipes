# Getting started

- open in the devcontainer
- `git clone https://github.com/cooklang/cooklang-import.git`

- cargo install --path .
- get openai API key on : https://platform.openai.com/api-keys
- export OPENAI_API_KEY
- `cooklang-import url`
- brew install nicholaswilde/tap/cook-docs
- cook-docs
